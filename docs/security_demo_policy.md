# 演示安全与测试说明

本项目面向数据库课程设计演示，保留本机默认配置，同时支持用环境变量覆盖敏感配置。

## 配置

- `SECONDHAND_DB_HOST`、`SECONDHAND_DB_PORT`、`SECONDHAND_DB_USER`、`SECONDHAND_DB_PASSWORD`、`SECONDHAND_DB_NAME` 可覆盖数据库连接。
- `SECONDHAND_SECRET_KEY` 可覆盖 Flask 会话密钥。
- `SECONDHAND_DEBUG=1` 可开启本机调试；默认关闭，避免部署时暴露调试信息。

## 验证码

当前不接入真实短信服务，注册、修改手机号、设置或修改支付密码统一使用测试验证码 `000000`。错误验证码会被拒绝。

## 上传

商品图片上传同时校验扩展名和文件头，支持 JPG、PNG、GIF、WEBP。伪装成图片扩展名的非图片文件会被拒绝。

## 钱包

钱包充值为站内余额功能，用户填写金额后增加账户余额，余额可用于订单支付；当前项目不接入第三方支付渠道。

## 验证脚本

- `_smoke.py` 是只读冒烟检查，只解析关键源码和模板，不连接或写入业务数据库。
- `_verify.py` 是端到端业务验证，会写入数据库测试数据，运行前应确认当前库允许被测试数据影响。
